Ability to change the design of login and registration page. Add additional inputs to registration page.

== Description ==

Customize your login and registration page.

You have the ability to change the logo as well as the background on the page.

You have the ability to add additional inputs to the registration page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ms-registration` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the MS Registration screen to configure the plugin
